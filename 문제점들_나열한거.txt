준성이의 문제점 

1. 코드의 묘듈화가 잘 안되어 있음.
    -> 난해함 
    -> 해결 방법은 파일을 더 만드는 것

    index.js -> main.js(합치는 곳), button.js(버튼 관련), screen.js(화면 관련)
    이렇게 분활시키면 됨
    또한 저 소괄호 표시는 git commit 할 때 쓰면 좋음 특히 제목으로
